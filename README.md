# oop-lab-credit-organization
Программа для учёта заемщиков кредитной организации. Реализует собственные структуры данных: очередь заемщиков на динамическом массиве со сдвигом и упорядоченный однонаправленный список платежей. Поддерживает добавление, поиск, изменение сумм кредита, работу с платежами, а также сохранение и загрузку данных.
